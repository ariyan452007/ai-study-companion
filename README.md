# AI Powered Study Companion

A full-stack, production-ready web application built to help students manage their study workflow, track topics, and leverage AI to automatically generate summaries and flashcards.

## Features
- **Dashboard Analytics**: Visualize study progress with interactive charts.
- **Subjects & Topics**: Hierarchical organization for all study materials.
- **Task Management**: Track deadlines, priorities, and revision schedules.
- **AI Tools**: Generate topic summaries and practice questions (Simulated in this repo via local state).

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- React Router DOM v6
- Recharts
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Deployment
This project is configured to be easily deployed to Vercel, Netlify, or GitHub Pages.
